# Подсказки по командной строке

Команда смены директории
```sh
cd c:\foider_name
``````

Листинг текущей лиректории
```sh
dir
```

Иницилизация 
```sh
init
```

Удаление файла
```sh
del
``````

Узнать статус (что происходит)
```sh
status
```

Добавить новое
```
add
````

Выбрать сохранение
```sh
checkout
```

Разница между текущей и фактической версией
```sh
diff
```

Комментировать
```sh
commit
```
# Отображение всех веток
```sh
git branch
``````
